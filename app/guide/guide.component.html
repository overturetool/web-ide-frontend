Read-Eval-Print-Loop

Evaluate VDM expressions here.
  • Expressions are evaluated against the currently open file.
  • $$ represents the latest result.
  • $n represents the n'th latest result. ($0 equals $$, $1 equals the 2nd latest result.)
  • Define a variable with := (n := 42 defines n and sets its value to 42.)
  • See the entire list of commands by evaluating &help.
